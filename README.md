LSP Userstatus
==============

Plugin for [LiveStreet CMS](http://livestreetcms.com/ "LiveStreet CMS")
-----------------------------------------------------------------------

Attention
To use this plugin requires **Livestreet CMS version 1.0 or later**.


About project
-------------
Добавляет простой текстовый статус в профиль пользователя
